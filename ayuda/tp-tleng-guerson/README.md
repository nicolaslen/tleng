# Trabajo Práctico de Teoría de Lenguajes

## Instrucciones


1. Instalar dependencias

```
make install
```

2. Ejecutar tests

```
make test
```

3. Ejecutar programa

```
./CLambda "<expresión entre comillas>"

ó

./CLambda 
(En este caso se deberá ingresar la expresión por stdin)
```

# Trabajo Práctico

Este trabajo práctico correspondiente a la materia Teoría de Lenguajes, tiene como objetivo implementar un compilador de cálculo lambda con extensiones Bool y Nat. 

La gramática utilizada es de tipo LALR y fue implementado en python utilizando la herramienta PLY.
