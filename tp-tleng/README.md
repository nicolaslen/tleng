# Trabajo Práctico de Teoría de Lenguajes

## Instrucciones


1. Instalar dependencias

```
make install
```

2. Ejecutar programa

```
./Json2Yaml "<expresión entre comillas>"

ó

./Json2Yaml 
(En este caso se deberá ingresar la expresión por stdin)
```

# Trabajo Práctico

Este trabajo práctico correspondiente a la materia Teoría de Lenguajes, tiene como objetivo implementar un convertor de JSON a YAML

La gramática utilizada es de tipo LALR y fue implementado en python utilizando la herramienta PLY.
